# Washing Task - Laravel Filament

Sistem manajemen tugas harian untuk cuci mobil menggunakan Laravel + Filament.

## Fitur
- Tambah, edit, hapus task
- Filter berdasarkan status
- Admin panel dengan Filament

## Instalasi
1. `git clone`
2. `composer install`
3. `cp .env.example .env`
4. `php artisan key:generate`
5. `php artisan migrate`
